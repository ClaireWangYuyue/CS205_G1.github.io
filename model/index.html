Model Details

Serial Version

  • Backtracking (brute force, depth first search):
    • depth first search
    • search through permutations of all possibly correct answers until it reaches a solution.
  • Pseudocode
    • At the i-th empty cell on the board:
        For all possible fill-ins:
         Fill the number
         If this is the last empty cell:
          Return the solution
         Else:
          Search the (i+1)-th empty cell
        Return with no solution

OpenMP Version

  • Serially generate a list of incomplete solutions
  • Perform backtracking algorithm in parallel by bootstrapping possible boards and adding them to a shared deque
  • Use dynamic scheduling to balance work load on all threads

MPI + OpenMP version

  • Serially generate a shorter list of incomplete solutions
  • Trunk the shorter list and dispatch to nodes.
    • Each node generate a longer list of incomplete solutions based on its own chunk
    • Each node perform OMP-level parallelism to solve the puzzle
  • All nodes output the solutions to a queue of solutions to be gathered and printed out by the master node
